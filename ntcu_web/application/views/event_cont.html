北一區第二梯次前導學校期末經驗分享座談會

為配合新課綱施行，國教署指出，107學年度將推動新住民語文課程前導計畫，分2梯次推動，前導學校遴選原則為邀請參與、指定參與及推薦參與，第一梯次自107年8月至108年1月，預計每縣市擇定1至3校，補助50校推行前導計畫。

明年8月上路的12年國教新課綱納入新住民語文，教育部國教署推動縣市新住民語文課程前導學校示範教學，107學年第1學期有21縣市50所國小及2所國中63班參與，開設7國語文課程。屏東縣恆春國小開設越南等3種語言最多，校長鄭鴻博昨天指出，校內學新住民語的大部分都不是「新二代」，且家長很支持。

12年國教課綱中，本土語文（含新住民語文）在國小必選、國中及高中則是選修。其中新住民語文以東南亞國家語文為主，包括越南、印尼、泰國、緬甸、柬埔寨、菲律賓及馬來西亞等7國官方語言。國小學生必須從本土語文及新住民語文中擇一必修，每周一節為原則；國中、高中階段則依學生需求於彈性課程開課。
