ABOUT

計畫簡介

為持續促進教學支援工作人員專業發展，精進新住民語文教學品質，提升教學效能，本計畫核心工作小組成員將協同主題領域專家學者、新住民語文課程開課學校之教學指導教師和教學支援人員、以及邀請縣市輔導團夥伴進行專業對話，期望透過健全地方教學輔導團，運作專業學習社群，評估語文程度級別等規劃，並朝整合運用資源，建立專業支持與輔導系統，爰委請國立臺中教育大學辦理「新住民語文課程教學實務精進推動計畫」，以四大主軸推動執行。

  • 學生選課歷程 (含文化向度、認知向度、情境向度)
  • 情境導向悅趣評估語文程度分級評估 (越南語第三冊)
  • 教材教法、媒體素養精進社群動態
  • 政策文宣、電子報推播

本計畫將逐步彙整有關新住民語文課程教育政策評估或擬定所需之實徵導向資料，以期提供教育部國教署於新住民語文課程教學、師資職能精進、以及政策宣導整體作為之參酌。
